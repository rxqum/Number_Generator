## Number Generator (Python + PyQt5)

A graphical number generaor created with Python and PyQt5.
It can generate a random number between 2 numbers (for example between 1 and 1000)

## FEATURES
- Generating a number between 2 numbers
- Handling errors (such as wrong input type)

## HOW TO RUN
1. Install Python 3.11.XX
2. Install PyQt5: 'pip install PyQt5
3. Download the project files
4. Run the main file: 'Number_Generator.py'
5. Now you can interact with the calculator and use it.
